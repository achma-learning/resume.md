# Les Antiviraux

> [!info] Enseignant : Pr. TASSI | Statut : 🔴 Brouillon → 🟢 Maîtrisé

## I. Introduction

- Les virus utilisent la machinerie cellulaire de l'hôte → cibles limitées sans toxicité cellulaire
- Principales cibles : synthèse ADN viral, ARN polymérase, protéase, intégrase, neuraminidase, fusion virale

## II. Antiviraux anti-herpétiques

### A. Aciclovir (Zovirax®) et Valaciclovir

| Paramètre | Détail |
|---|---|
| Mécanisme | Prodrug phosphorylé par thymidine kinase virale → inhibition ADN polymérase virale (sélectivité ++) |
| Spectre | HSV1, HSV2, VZV (+ faible activité EBV, CMV) |
| Voie | Oral (valaciclovir = prodrogue = meilleure biodisponibilité), IV (formes graves) |
| EI | Néphrotoxicité (cristallurie → hydratation ++), neurotoxicité IV |
| CI | IR sévère sans adaptation |

**Indications :**
- Primo-infection herpétique (HSV), récurrences
- Zona (VZV) : valaciclovir 3 × 1g/j × 7j
- Encéphalite herpétique : Aciclovir IV **urgence** (10 mg/kg × 3/j × 14-21j)
- Prévention chez immunodéprimé

### B. Ganciclovir / Valganciclovir (anti-CMV)

- Indication : **CMV** (rétinite CMV du VIH, infections CMV de l'immunodéprimé)
- EI : **myélosuppression** (neutropénie, thrombopénie) +++, tératogène → contraception obligatoire

### C. Foscarnet

- Indication : CMV résistant au ganciclovir, HSV aciclovir-résistant
- EI : néphrotoxicité, hypocalcémie, hypomagésiémie

## III. Antiviraux anti-VIH (ARV)

### A. Classes

| Classe | Médicaments représentatifs | Cible |
|---|---|---|
| INTI (Inhibiteurs nucléosidiques transcriptase inverse) | Ténofovir (TDF), Emtricitabine (FTC), Lamivudine (3TC), Abacavir (ABC), Zidovudine (AZT) | Transcriptase inverse (incorporés dans ADN viral) |
| INNTI (Inhibiteurs non nucléosidiques) | Efavirenz (EFV), Névirapine, Rilpivirine | Transcriptase inverse (non compétitifs) |
| IP (Inhibiteurs de protéase) | Lopinavir, Atazanavir, Darunavir | Protéase VIH |
| INI (Inhibiteurs d'intégrase) | Raltégravir, Dolutégravir, Bictégravir | Intégrase VIH |
| Inhibiteurs de fusion | Enfuvirtide (T20) | Glycoprotéine gp41 |
| Antagonistes CCR5 | Maraviroc | Co-récepteur CCR5 |

### B. Principes de la trithérapie (HAART)

- **Toujours en association** (au moins 3 agents, combinant 2 classes) → prévenir émergence résistances
- Traitement à vie
- Combinaisons préférées actuelles : **2 INTI + 1 INI** (ex : TDF + FTC + Dolutégravir)

### C. Effets indésirables notables des ARV

| Médicament | EI spécifique |
|---|---|
| Ténofovir (TDF) | Néphrotoxicité, ostéopénie |
| Abacavir | Réaction d'hypersensibilité (porteurs HLA-B*5701) |
| Efavirenz | Troubles neuropsychiques (cauchemars, dépression) |
| Zidovudine | Anémie, myopathie |
| IP | Dyslipidémie, lipodystrophie, diabète |

## IV. Antiviraux anti-VHC (Hépatite C)

### Antiviraux à action directe (AAD)

- **Révolution thérapeutique** depuis 2014 : guérison > 95% en 8-12 semaines
- Combinaisons : **Sofosbuvir + Lédipasvir**, **Glécaprévir + Pibrentasvir**, etc.
- Mécanismes : inhibiteurs NS5B (polymérase), NS5A, NS3/4A (protéase)
- Tolérance excellente (EI minimes)

---

## V. Antiviraux anti-grippe

| Médicament | Mécanisme | Indication |
|---|---|---|
| **Oseltamivir** (Tamiflu®) | Inhibiteur neuraminidase → blocage libération virions | Grippe A et B (dans 48h du début) |
| Zanamivir (inhalé) | Idem | Grippe résistante à oseltamivir |
| Baloxavir | Inhibiteur de l'endonucléase virale | Grippe (alternative) |

---

## VI. Antiviraux anti-VHB

| Médicament | Particularité |
|---|---|
| **Ténofovir** (TDF, TAF) | 1ère ligne, très puissant, peu de résistances |
| **Entécavir** | 1ère ligne, puissant |
| Lamivudine | Résistance fréquente, moins utilisé |
| Interféron α pégylé | Traitement court (48 semaines), taux séroconversion limité |

## Zone de révision active

> [!question] Questions
> **Q1** : Quel est le mécanisme de sélectivité de l'aciclovir ?
> **R1** : Il est phosphorylé spécifiquement par la thymidine kinase virale (codée par le virus herpétique) → actif uniquement dans les cellules infectées.
>
> **Q2** : Quelle est la trithérapie de référence actuelle contre le VIH ?
> **R2** : 2 INTI (ex: TDF + FTC) + 1 INI (ex: dolutégravir).

> [!success] Points tombables ⭐
> - Aciclovir : mécanisme de sélectivité (TK virale)
> - Encéphalite herpétique → Aciclovir IV urgence
> - Ganciclovir → CMV, myélosuppression majeure
> - VIH : toujours trithérapie, 2 INTI + INI
> - Oseltamivir : dans les 48h de la grippe
> - AAD : guérison VHC > 95%
