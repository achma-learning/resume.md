# Les Quinolones / Fluoroquinolones

> [!info] Enseignant : Pr. TASSI | Statut : 🔴 Brouillon → 🟢 Maîtrisé

## I. Généralités

- **Mécanisme** : Inhibition de l'**ADN-gyrase** (topoisomérase II, Gram-) et de la **topoisomérase IV** (Gram+) → blocage réplication et transcription de l'ADN bactérien → bactéricide
- **Effet** : Bactéricide, **concentration-dépendant** (AUC/CMI ≥ 125 ou Cmax/CMI ≥ 10)
- **Avantage majeur** : excellente **biodisponibilité orale** (70-100%) = efficacité similaire IV et orale

## II. Classification

| Génération | Médicaments | Spectre |
|---|---|---|
| 1ère (quinolones) | Acide nalidixique | Gram- urinaires uniquement (retraite) |
| 2ème (FQ classiques) | Ciprofloxacine (Cipro®), Ofloxacine | Gram- large, Pseudomonas, intracellulaires |
| 3ème (FQ respiratoires) | Lévofloxacine (Tavanic®) | + Gram+ (pneumocoque), légionelle |
| 4ème | Moxifloxacine | + anaérobies, **pas Pseudomonas**, antituberculeux |

## III. Spectre

| Bactéries | Couvertes par |
|---|---|
| Entérobactéries (E. coli) | Toutes |
| Pseudomonas aeruginosa | Ciprofloxacine, Lévofloxacine |
| Staphylocoques | Ciprofloxacine (SASM uniquement), Lévofloxacine |
| Pneumocoque | Lévofloxacine, Moxifloxacine |
| Bactéries intracellulaires (Legionella, Chlamydia, Mycoplasma) | Toutes FQ |
| Anaérobies | Moxifloxacine |
| Mycobacterium tuberculosis | Lévofloxacine, Moxifloxacine (2ème ligne TB) |

## IV. Pharmacocinétique

- Très bonne diffusion tissulaire : **prostate** (ciprofloxacine +++), os, poumon, LCR (lévofloxacine)
- Élimination : rénale (ciprofloxacine, lévofloxacine) → adapter à l'IR
- Moxifloxacine : élimination hépatique-biliaire → **pas d'adaptation en IR**

## V. Effets indésirables

> [!danger] Effets indésirables importants
> 1. **Tendinopathies / rupture tendineuse** : surtout tendon d'Achille → **arrêt immédiat si douleur tendineuse** ; risque ↑ avec corticoïdes, sujet âgé
> 2. **Allongement QT** : risque de torsade de pointes → **CI** si QT long, avec autres médicaments allongeant QT
> 3. **Arthropathie / cartilage** → **CI chez < 18 ans** (cartilages de croissance)
> 4. **Neuropathie périphérique** (rare mais irréversible)
> 5. **Troubles neuropsychiques** : insomnie, anxiété, convulsions (épileptiques ++)
> 6. **Photosensibilisation** : éviter exposition solaire
> 7. **Sélection de résistances** : inducteur majeur de résistances → limiter les prescriptions

## VI. Contre-indications

- < 18 ans (sauf mucoviscidose à Pseudomonas)
- Grossesse (arthropathie fœtale, allaitement)
- Épilepsie (abaissement seuil convulsif)
- QT long ou médicaments allongeant QT
- Allergie

## VII. Interactions médicamenteuses

- **Antiacides, pansements gastriques, fer, calcium** → chélation → ↓ absorption orale (prendre 2h avant ou 4h après)
- **AINS** → risque convulsions
- **Médicaments allongeant QT** → CI (amiodarone, antiarythmiques classe IA/III)
- **Warfarine** : ciprofloxacine ↑ effet anticoagulant

## VIII. Indications

| Infection | FQ préférée |
|---|---|
| Infections urinaires hautes (pyélonéphrite) | Ciprofloxacine ou Lévofloxacine |
| Pneumonie atypique / légionellose | Lévofloxacine ou Moxifloxacine |
| Prostatite bactérienne | **Ciprofloxacine** (diffusion prostatique ++) |
| Infections à Pseudomonas | Ciprofloxacine IV |
| Gonorrhée résistante | Ciprofloxacine (si sensible) |
| Tuberculose résistante | Lévofloxacine, Moxifloxacine |
